# dataset_Rabi_Model
This is a dataset for paper "Probing Liouvillian Spectrum and Dynamical Crossover in Open Quantum Rabi Model".

**Fig. 2:** The experimental data points are given by (`tlist_exp`, `phon_exp`), while the simulated curve is given by (`tlist`, `phon`). The steady-state value is denoted as `phon_ss`.

**Fig. 3a:** Experimental spectrum shown as (`freq_exp`, `reSpec_exp`).

**Fig. 3b:** Experimentally extracted Liouvillian modes (`alpha_exp`, `beta_exp`), with corresponding errors (`sig_alpha`, `sig_beta`).

**Fig. 4a:** Decay of the characteristic function for coupling strengths `gcr_Fig4_tilde(i)`. The horizontal axis is `tlist`, and the vertical axis is `f_decay_exp_Fig4a{i}`.

**Fig. 4b:** Simulated curve (`gcr_tilde`, `eta`). Experimental points (`gcr_Fig4_tilde`, `eta_exp_Fig4b`) with error bars `Err_eta_Fig4b`.
